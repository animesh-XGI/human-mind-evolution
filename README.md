# # human-mind-evolution

A structured research log on human evolution through Equation 3 (E3) — the brain-culture feedback loop that gives rise to mind, agency, and self-modelling intelligence.

## Purpose

This repository exists to develop, test, and refine a coherent framework for understanding human evolution, with Equation 3 (E3) as the current primary lens.

The aim is not only to collect notes, but to build a durable research system that separates:
- framework-level thinking,
- chronological session logs,
- concept development,
- source tracking, and
- evolving models and hypotheses.

## Current focus

- Equation 3 (E3)
- Brain-culture feedback loop
- Human mind and agency
- Self-modelling intelligence
- Evolutionary continuity vs discontinuity

## Working method

This repository is intended to evolve session by session.

General method:
- Use `02-sessions/` for chronological thinking and session capture
- Use `01-frameworks/` for formal equation-level work
- Use `03-concepts/` for developing individual ideas cleanly over time
- Use `04-sources/` for references without mixing them into interpretation
- Use `05-models/` for hypotheses, causal loops, diagrams, and open questions
